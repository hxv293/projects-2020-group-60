# Model analysis project

In this project we have choosen to analyze a simple two period economic model with two different social security systems, more specifically the pay-as-you-go system and the fully funded system and compared the Capital accumulation between the two types. The before mention models is found in MacroIII exam Winter 2019. 

The **results** of the project can be seen from running [modelproject.ipynb](modelproject.ipynb).

**Dependencies:** Apart from a standard Anaconda Python 3 installation, the project requires no further packages.
